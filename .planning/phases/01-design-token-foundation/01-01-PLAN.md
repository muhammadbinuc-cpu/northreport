<objective>
Rewrite the globals.css design token system from dark glassmorphism to the NorthReport brand palette (maroon/cream/charcoal) and update all CSS component classes to use the new light-mode tokens.

Purpose: This is the foundation layer that every subsequent phase depends on. Changing :root variables propagates the new palette to all 45+ components automatically through their existing var() references. Typography classes, shadow values, status colors, and component-level CSS classes all need updating to work correctly on light backgrounds.

Output: A fully restructured globals.css with three-layer token architecture + updated animations.css with .glass-card removed.
</objective>
